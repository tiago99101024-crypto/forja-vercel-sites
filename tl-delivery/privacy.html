Política de Privacidade

Esta página descreve o uso de dados para o GPT FORJA Meta MCP LAB.

As ações disponíveis usam dados de laboratório para teste técnico. Nenhuma conta real de anúncio é consultada.

As respostas da ação podem retornar identificadores, nomes e saldos fictícios usados apenas para validação.
